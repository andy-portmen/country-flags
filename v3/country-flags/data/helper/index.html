One Last Step

To enable the extension to connect with native applications, you'll need one additional step: download, unzip, and install the open-source NodeJS native client.

  • Click here to download the package for you OS (Window, Mac, and Linux are currently supported) or click here to browse all available download options.
  • Extract the downloaded package in a local directory
  • Now open a terminal in the root directory and run ./install.sh
  • Now double-click the install.bat file

The installer script places a few files in two different locations so that your browser can recognize the native client. If NodeJS is already installed on your system, that version will be usedâ€”so please ensure it's up to date. If NodeJS is not installed, the portable version included with the client will be used instead. You can learn more on andy-portmen/native-client's GitHub page.

To completely remove this native client, run ./uninstall.shuninstall.bat.

To verify that the native client is accessible, click the "Check Connection" button. If you still see this page even though the check button reports a successful connection, it means the application cannot be reached (the path may be incorrect in the options page). If the executable path is correct and the client is connected, you can troubleshoot further by enabling console logs in Chrome, Edge or Opera, or in Firefox.
